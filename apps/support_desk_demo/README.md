# support_desk_demo

고객 문의 인박스를 위한 작은 demo app입니다.

목표는 처음부터 agent를 추상적으로 설계하는 것이 아니라, 사람이 실제로 고객 문의를 처리하는 제품 표면을 먼저 만드는 것입니다. agent는 이 흐름의 특정 지점에 제안을 제출합니다.

첫 단계에서는 백엔드만 제공합니다.

```text
고객 문의 목록
-> 고객 문의 상세
-> 상태 변경
-> 내부 메모
-> 응답 초안
-> agent proposal 수신
-> 사람이 proposal 검토
```

프론트엔드는 이후 이 API를 바탕으로 붙입니다.
